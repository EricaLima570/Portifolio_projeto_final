HTML

HTML é a base de qualquer página web. Com ele, conseguimos estruturar o conteúdo da página. Veja um exemplo básico:

<div class="container">
    <h1>Olá, mundo!</h1>
    <p>Este é meu primeiro site usando HTML!</p>
</div>

Dica: Ao usar o Bootstrap, você pode utilizar as classes para facilitar a estruturação do layout. A classe container ajuda a centralizar o conteúdo.

CSS

CSS é usado para estilizar e formatar a aparência da sua página. Com ele, conseguimos deixar as páginas visualmente mais atraentes. Veja um exemplo de como aplicar um fundo e centralizar o texto:

body {
    background-image: url('imagem-de-fundo.jpg');
    background-size: cover;
    text-align: center;
    color: #fff;
}

Dica: Para centralizar elementos de forma eficiente, você pode usar a classe text-center do Bootstrap, ou utilizar propriedades de flexbox ou grid no CSS.

Bootstrap

Bootstrap é uma excelente ferramenta para agilizar o desenvolvimento web. Ele oferece classes prontas para criar layouts responsivos e componentes como botões, formulários, etc. Veja um exemplo de um formulário simples:

<form>
    <div class="form-group">
        <label for="nome">Nome</label>
        <input type="text" class="form-control" id="nome" placeholder="Digite seu nome">
    </div>
    <button type="submit" class="btn btn-primary">Enviar</button>
</form>

Dica: O Bootstrap também tem várias classes de utilidade como btn-primary e form-control que tornam o código mais limpo e organizado.

Versionamento com Git

Git é uma ferramenta essencial para o versionamento de código. Ele ajuda a manter o controle das mudanças no código e facilita o trabalho em equipe. Abaixo, temos alguns comandos básicos:

# Inicializa um repositório git
git init

# Adiciona arquivos ao stage
git add .

# Comita as alterações
git commit -m "Mensagem do commit"

# Verifica o status do repositório
git status

Dica: Sempre faça commits pequenos e frequentes. Isso facilita o controle de alterações e a reversão de mudanças indesejadas.

Dicas Finais

  • Utilize display: flex, justify-content e align-items no CSS para centralizar elementos facilmente.
  • Adicione bordas personalizadas usando border-radius, box-shadow e cores com transparência (rgba).
  • Defina imagens de fundo usando background-image, combinando com background-size: cover e background-attachment: fixed.
  • Estilize textos de seções específicas usando seletores de CSS e altere a propriedade color.
  • Mantenha o CSS em arquivos separados para melhor organização e reutilização do código.
  • Crie seções estilizadas com fundos transparentes para destacar informações importantes.
  • Adicione efeitos interativos em elementos com o seletor :hover.
  • Destacar ícones com bordas arredondadas e escalas interativas utilizando border-radius e transform: scale().
  • Para centralizar caixas de conteúdo na tela, use display: flex no contêiner pai.
  • Implemente o Bootstrap para criar layouts responsivos com componentes prontos.
  • Adicione profundidade aos elementos aplicando box-shadow.
  • Personalize links com transições suaves utilizando transition e estilos específicos para :hover.

Lembre-se: a prática constante e a exploração de novas ideias são fundamentais para se tornar um desenvolvedor web melhor!
